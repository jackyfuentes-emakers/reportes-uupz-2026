# Glosario UUPZ

## Términos de negocio
| Término | Significado |
|---------|-------------|
| vasos | Unidades de producto vendidas (vasos/termos UUPZ) |
| pedidos / paquetes | Número de órdenes o envíos |
| neto | Ingreso neto = bruto × (1−comisión) / 1.16 (IVA) |
| bruto | Ingreso bruto antes de deducciones |
| ticket | Ticket promedio = bruto total / pedidos |
| PV neto / $/vaso | Precio promedio por vaso = neto / vasos |
| YoY | Year-over-Year: comparativa vs mismo periodo año anterior |
| MoM | Month-over-Month: comparativa vs mes anterior |
| DOW | Day of Week: distribución de ventas por día de la semana |

## Términos de reporte
| Término | Significado |
|---------|-------------|
| SEM / SEM10 | Semana del año (SEM10 = semana 10 del año) |
| PLAT | Array JavaScript con configuración de las 5 plataformas |
| ACTIVE | Subconjunto de PLAT con `active:true` (para filtros) |
| KPI pill | Chip/badge en el header con métricas clave |
| stacked bar | Gráfica de barras apiladas (desglose modelo×capacidad) |
| canvas | Elemento HTML5 Canvas donde se dibujan las gráficas |
